There are two ways to check for equality in JavaScript

  1. double equals => relational operator that returns true iff both operands have the same value. Type coersion is performed
  2. triple equals => relational operator that returns true iff both operands are of the same type and same value. It checks for type equality first. (!== exists too)

Truthy and Falsey

Falsey refers to any value that is not the boolean false, however it evaluates to false.

Values that are Falsey include:

  • The empty string
  • number 0
  • null and undefined
  • and NaN

for undefined and NaN, only another undefined and NaN respectively can be equal to each other

every other value is truthy.

we can use the typeof operand to determine the type of something.
