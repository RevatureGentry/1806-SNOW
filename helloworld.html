  • boolean

The other data types inclue but are not limited to

  • Object
  • Function
  • Array
  • Date
  • Error

There are two ways to check for equality in JS

  1. Double equals returns true if and only if both operands have the same value type coercion is performed
  2. Triple equals returns true if and only if both are same type and value

Truthy and Falsy

"Falsy" refers to any value that is not the boolean false, however it evaluates to false

Values that are "Falsy" include

Null and undefined are not equal to anything but themselves

NaN is not equivalent to anything

every other value is truthy

we can use the typeof operator to determine the type of its operand



The different variable scope in JavaScript are

  • Global
  • Local
  • Block

Functions in Javascript are special

  • They can be assigned in one of three ways: declared function, function as a value, arrow functions
  • They are objects
  • They can be anonymous
  • They can be hoisted
  • They can be self invoking
  • They can be used as callback fucntions


New in EcmaScript 6 (ES6)

  • Template Literals: strings crated using the ES6 backticks, ``, which allow for the interpolation of JavaScript expressions inside strings
  • Arrow Functions: provide shorter syntax than the Declaration of a Function. Arrow functions are necessary because they inherit the 'this' value of its enclosing context
