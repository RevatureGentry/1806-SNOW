There are two ways to check for equality in JavaScript

  1. Double Equals => relational operator that returns true if and only if both operands have the same value. Type coersion is performed (There also !=)
  2. Tripe Equals => relational operator that returns true if and only if both operands are of the same type AND have the same value. It checks type equality first (There also !==)

Truthy and Falsy

"Falsy" refers to any value that is not the boolean false, however it evaluates to false

Values that are "Falsy" include

  • The empty string ''
  • The number 0
  • null and undefined
  • NaN

Null and undefined are not equal to anything but themselves

NaN is not equivalent to anything, including itself

Every other value is truthy

We can use the 'typeof' operator to determine the type of its operand
