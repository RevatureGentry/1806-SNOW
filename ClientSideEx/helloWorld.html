There are two ways to check for equality in JavaScript

  1. Double Equals => Relational operator that returns true if and only if both operands have the same value. Type Coersion is pefromed (There is also !=)
  2. Tripple Equals => Relational operator returns true if and only if both operands are of the same type AND have the same value. It checks for type equality first. (There is also !==)
