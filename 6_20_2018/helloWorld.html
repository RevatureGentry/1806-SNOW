Let's do some scope stuff

Functions in JavaScript are special

Can be called in 3 ways:

  • ARE OBJECTS
  • SELF INVOKING
  • CALLBACK FUNCTIONS
  • HOISTED
  • ANONYMOUS

New in ECMAScript 6 (ES6)

  • Template literals: strings created using ES6 backticks, ``, which allow for interpolation of JavaScript expression inside strings
  • Arrow Functions: provide a shorter syntax than the declaration of a function. Arrows Functions, however, are necessary because they inherit the 'this' value of its enclosing context
