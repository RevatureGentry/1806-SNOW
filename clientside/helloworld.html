The primitive types in JS are:

  • Number - A 64 bit floating point
  • Boolean
  • String
  • Undefined
  • Symbol

The other data types are

  • Object
  • Function
  • Array
  • Date
  • Error

There are two ways to check for equality in Javascript

  1. Double Equals - Relational operator that returns true if and only if both operands have the same value. There is also !=. Type coercion is performed.
  2. Triple equals is a relational operator that returns true if and only if both operands of the same type and value. There is also !==.

Falsey refers to any value that is not the boolean false, however, it evaluates to false.

Values that are falsey include

  • Empty string
  • Number 0
  • Null
  • Undefined
  • NaN

Null and undefined are not equal to anything but themselves.

NaN is not equivalent to anything, including itself.

Every other value is truthy.

We can use the typeof operator to determine the type of its operand
