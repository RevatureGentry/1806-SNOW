The primitive types in JavaScript are:

  • number - a 64 bit floating point
  • boolean
  • string
  • null
  • undefined
  • symbol

The other data types include but are not limited to:

  • Object
  • Function
  • Array
  • Date
  • Error

There are two ways to check for equality in JavaScript

  1. Double Equals => Relational operator that returns true iff both operands have the same value. Type coercion is performed. (There also is !=)
  2. Triple Equals => Relational operator that returns true iff both operands have the same type AND the same value. It checks for type equality first, so type coercion is not performed. (There also is !==)

Truthy and Falsy

"Falsy" refers to any value that is not the boolean false, however it evaluates to false.

Values that are falsy include:

  • The empty string ""
  • The number zero
  • null and undefined
  • NaN

Nul and undefined are not equal to anything but themselves

NaN is not equivalent to anything, including itself

Every other value is truthy

We can use the 'typeof' operator to determine the type of its operand



The different variable scopes in JavaScript are:

  • Global
  • Local
  • Block

Functions in JavaScript are special.

  • They can be assigned in one of three ways: declared function, function as a value, arrow functions
  • They're objects
  • They can be anonymous
  • They can be hoisted
  • They can be self invoking
  • They can be used as callback functions

New in EcmaScript 6 (ES6)

  • Template Literals: strings created using the ES6 backticks, ``, which allow for the interpolation of JavaScript expressions inside strings.
  • Arrow functions: provide a shorter syntax than the declaration of a function. Arrow functions are necesary because they inherit the 'this' value of its enclosing context
