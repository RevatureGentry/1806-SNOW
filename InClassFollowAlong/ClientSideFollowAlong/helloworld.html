The primitive types in JavaScript are

  • number
  • boolean
  • string
  • null
  • undefined
  • symbol

The other data types include, but are not limited to:

  • object
  • function
  • array
  • date
  • error

There are two ways to check for equality in JavaScript

  1. Double equals => relational operator that returns true if and only if both operands have the same value. Type coercion is performed.
  2. Triple equals => relational operator that returns true if and only if both operands are of the same type

Truthy and Falsy

"Falsy" refers to any value that is not the boolean false, however it evaluates to false

Values that are "Falsy" include:

  • The empty string
  • The number 0
  • null and undefined
  • NaN

Null and undefined are not equal to anything but themselves

NaN is not equivalent to anything, including itself

Every other value is truthy
