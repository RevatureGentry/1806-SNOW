The primitive types in JavaScript are

  • Boolean
  • Number
  • String
  • Null
  • Undefined
  • Symbol

Other Data Types include, but are not limited to

  • Object
  • Function
  • Array
  • Data
  • Error

There are two ways to check for equality in JavaScript

  1. Check if values are equal, (==)
  2. Check if values AND type are equal, checks type equality first (===)
  3. Negator (!=)

Truthy and Falsy

Falsy refers to any value that is not the boolean false, however it evaluates to false

Ex: empty string, number zero (0), null, undefined, NaN

All other values are considered truthy

NaN and null are NOT THE SAME THING

We can use the "typeof" operator to determine the type of its operand
