There are 2 ways to check for equality in JavaScripts

  1. Double Equals => Relational operators that returns true if and only if both operands have the same value. Type coersion is performed. (There also !=)
  2. Triple Equals => Relational operators that returns true if and only if both operands are of the same type and same value. It checks equality first. (There also !==)

Truthy and Falsy

"Falsy" refers to any value that is not the boolean flase, however it evalulates to false

Values that are "Falsy" includes

  • The empty string
  • the number 0
  • null and undefined
  • NaN (Not a Number)

Null and undefined are not equal to anything but themselves

NaN is not equivalent to anything, including itself

Every other value is truthy

We can use the 'typeof' operator to determine the type of its operand



The different variable scopes in JavaScripts are

  • Global
  • Local

Function in JavaScripts are specials

  • They can be assigned in one of three ways: decleared function, function as a value, arrow function
  • They are Objects
  • They can be anonymous
  • They can be hoisted
  • They can be self invoking
  • They can be used as callback function

New in EcmaScript 6 (ES6)

  • Template Literals: strings created using ES6 backticks, ``, which allow for the interpilation of JavaScript expressions inside strings
  • Arrow Functions: provides shorter syntax than the Declaration of a Function. Arrow functions are necessary because they inherit the 'this' value of its enclosing content.
