The primitive types in JavaScript are:

  • number - a 64 bit floating point
  • boolean
  • string
  • null
  • undefined
  • symbol

The other data types include, but are not limited to:

  • Object
  • Function
  • Array
  • Date
  • Error

There are two ways to check for equality in JavaScript

  1. Double Equals => Relational operator that returns true if and only if both operands have the same value. Type coersion is performed. (Also !=)
  2. Triple Equals => Relational operator that returns true if and only if both operands are of the same type and have the same value. Checks for type equality first. (Also !==)

Truthy and Falsy

"Falsy" is any value that is not the boolean false, however it evaluates to false.

Values that are "Falsy" include:

  • The empty string
  • The number 0
  • null and undefined
  • NaN

Null and undefined are not equal to anything but themselves

NaN is not equivalent to anything, including itself

Every other value is truthy

We can use the 'typeof' operator to determine the type of its operand
