You can include labels for any input. for='id_name'.
This allows you to click on the actual label to select the input box.


How would you like your pizza delivered?



To link these radio buttons together, they need to all have the same value for the 'name' attribute.
You link together a label, such as 'Teleporter', and an input, the radio button, by matching up the 'for=' attribute in label with the ''id=' attribute in input.
The 'value=' attribute is what is returned/sent.

To make sure certain elements of the form must be filled with values before submitting, add the 'required' attribute to any necessary inputs.
